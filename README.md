# My Portfolio

This is my personal portfolio website, showcasing my projects, skills, and contact information. The application is built using **React** for the front end, styled with **Tailwind CSS**, and uses **React Router** for seamless routing between pages.

## Features

- **Responsive Design**: The portfolio is fully responsive and adapts to different screen sizes.
- **Routing**: Smooth navigation between pages using React Router.
- **Tailwind CSS Styling**: Easy and fast styling with a utility-first CSS framework.
- **Showcasing Projects**: Includes sections for my projects, skills, and contact details.

## Technologies Used

- **React**: JavaScript library for building user interfaces.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **React Router**: Declarative routing for React applications.

## Setup and Installation

Follow these steps to set up and run the project on your local machine:

### Prerequisites

Ensure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js) or yarn (optional)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. Navigate to the project directory:

   ```bash
   cd My_Portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

### Running the Project

1. Start the development server:

   ```bash
   npm start
   ```

2. Open your browser and navigate to:

   ```
   http://localhost:5173
   ```

### Building the Project

To build the project for production:

```bash
npm run build
```

The optimized build will be available in the `build` folder.
